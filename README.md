## Features

* Safe navigation with route display
* AI-based risk prediction (demo logic)
* Local companion listing
* Nearby help (police stations and hospitals)
* Emergency SOS alert system
* Clean and modern user interface

* Flutter
* Dart
* OpenStreetMap using flutter_map
